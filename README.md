# 🧀 치즈 헌터 (Cheese Hunter)

**한글 퀴즈와 뱀 게임이 결합된 아케이드 게임!**  
빠르게 정답을 맞히고, 더 많은 치즈를 모아 꼬리를 늘리세요! 🌟

---

## 🎮 프로젝트 소개

**‘치즈 헌터’**는 고전적인 **Snake Game**의 조작 방식에  
**한글 초성 퀴즈 시스템**을 결합한 웹 기반 아케이드 게임입니다.  

단순한 조작을 넘어 **순발력 + 어휘력**을 동시에 요구하며,  
학습적 재미와 아케이드 감성을 함께 즐길 수 있습니다.

---

## 🕹️ 게임 방법 및 규칙

| 요소 | 설명 |
|------|------|
| **조작** | 화살표 키 또는 **WASD** 키로 뱀(쥐)을 조작합니다. |
| **치즈 획득** | 🧀 치즈에 닿으면 **한글 초성 퀴즈** 창이 열립니다.<br>힌트와 초성만 보고 정답을 입력해야 꼬리가 늘어나고 점수를 얻습니다. |
| **콤보 시스템** | 연속으로 정답을 맞히면 **콤보 배율**이 적용되어 점수가 폭발적으로 증가합니다. |
| **일시정지** | 게임 중 **Spacebar** 또는 **P** 키를 눌러 일시정지할 수 있습니다. |

## 🧩 퀴즈 데이터 구조 설명

‘치즈 헌터’의 퀴즈는 **JSON 형태의 배열**로 관리됩니다.  
각 퀴즈 객체는 **초성, 정답, 힌트** 세 가지 속성을 가집니다.

```js
const quizData = [
  {
    question: "ㅅㅍ",     // 초성
    answer: "사과",       // 정답
    hint: "빨갛고 달콤한 과일"  // 힌트
  },
  {
    question: "ㅂㄹㅈ", 
    answer: "바나나", 
    hint: "노란색 열대과일"
  },
  {
    question: "ㅎㅁ", 
    answer: "호미", 
    hint: "밭을 가꾸는 데 쓰는 도구"
  }
];
```

> ✅ **확장성**  
> - JSON 또는 외부 파일(`quizData.json`)로 분리하여 관리할 수 있습니다.  
> - 난이도별(`easy`, `normal`, `hard`) 퀴즈 세트를 추가하여 다양한 모드 구성도 가능합니다.
